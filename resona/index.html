核心价值

为什么是 Resona

groups
人物，是产品的灵魂

市面上的 AI 写作工具都在比"写得通顺"。我们比的是"写得像谁"——预置人物、原创人格、用户自建私有人格， 每一支笔都是有指纹的。

tune
可控的不确定性

风格滑块 + 多候选 + 与原文的 diff 视图，把"AI 生成"变成"我在挑"。 创作权牢牢握在用户手里，AI 只是替你试穿不同声音的草稿。

graphic_eq
从单段一路到播客

同一份草稿，可以是朋友圈一句话，也可以是一档双人对谈。 改写、扩写、播客脚本、配音导出——一条工作流贯通，无须切换工具。

Design System

一支会"换笔触"的设计语言

Palette
暮色紫 为主——创作、想象、有人格感的颜色。 玫粉 为播客流的语义色，区分"声音"分支。 深墨用于文字与人物徽章，避免紫色喧宾夺主。
Typography
借他人之笔
Inter · Noto Serif SC
标题与改写结果用思源宋体，让"成稿文字"有印刷质感；UI 与说明用 Inter， 保持工具感与现代清爽。中英混排时刻意拉开字重对比。
Principles
  • looks_one以人物为入口，不是以功能
  • looks_two样张胜过描述，所见即所得
  • looks_3三候选 + 滑块 = 可控的不确定性
  • looks_4永远显式标注"以 XX 风格生成"
One Line
「不是帮你写得通顺的 AI，
是帮你写得像"那个人"的笔。」
— 笔替 Resona 产品一句话
