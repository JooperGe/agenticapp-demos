## 一、产品概述

### 背景与痛点

现代人（尤其18-35岁城市青年）面临巨大的心理压力，但现有解决方案存在明显断层：

- **专业心理咨询**：贵（300-1000元/次）、难预约、病耻感强
- **熟人倾诉**：担心被评判、怕给朋友带来负担、社交关系压力
- **陌生人社交App（Soul等）**：噪音多、荷尔蒙驱动、不聚焦情绪疏导
- **AI陪伴产品**：缺乏真人温度，"被理解"的感受不足
- **公益热线**：等待久、体验差、无产品化

核心问题：**用户不一定需要"治疗"，更多需要"被听见"。** 在专业咨询和随便聊聊之间，缺少一个"轻量、安全、有温度的倾诉出口"。

### 产品定位

> 深夜情绪的安全出口——匿名、即时、有人听

**目标用户**：18-35岁城市青年，有情绪压力但未达到需要专业干预的程度

**一句话描述**：匹配一个经过培训的陌生倾听者，用文字陪你度过难熬的时刻

**技术形态**：微信小程序（MVP）→ 独立App（增长期）

## 二、竞品分析

| 产品 | 类型 | 优势 | 劣势 |
|------|------|------|------|
| 一罐（已停运） | 匿名树洞社区 | 验证了匿名情绪需求；冷启动极快 | 死于"想做大"——从树洞变社交，老用户流失 |
| Soul | 陌生人社交 | 体量大 | 不聚焦心理疏导，社交噪音大 |
| 倾伴心理 | 付费真人咨询 | 专业 | 按分钟收费，门槛高 |
| 测测AI / 星云星空 | AI情感陪伴 | 24h在线，成本低 | 缺乏真人温度 |
| Wysa | AI+CBT工具 | 循证方法 | 海外产品，无中文，AI非真人 |

### 竞品空白与机会

在"匿名 + 中等专业度（培训过的真人倾听者）"的象限，目前没有成功产品占据。一罐验证了需求成立但死于方向偏移，倾伴太贵太重，Soul太吵。

### 一罐的核心教训

1. **十倍目标是万恶之源**：为了融资讲大故事，偏离了验证成立的核心体验
2. **坚守窄定位**：不要为增长把产品变成"陌生人社交"
3. **小而美可持续 > 做大后猝死**：50万日活可能就是天花板，接受它
4. **先验证付费意愿**：再考虑是否需要融资

## 三、核心功能设计

### 3.1 用户核心流程

```
【倾诉者旅程】
打开小程序
    │
    ▼
选择当前情绪标签（焦虑 / 低落 / 愤怒 / 孤独 / 迷茫 / 压力）
    │
    ▼
匹配倾听者（15-60秒）
    │
    ├─ 匹配成功 → 进入文字对话（限时20分钟）
    └─ 无人在线 → AI倾听者兜底（明确告知是AI）
    │
    ▼
对话结束
    │
    ├─ 情绪评分（对话后 vs 对话前）
    ├─ 匿名感谢卡（可选）
    └─ 推荐自助工具（呼吸练习 / 情绪日记入口）
```

### 3.2 功能模块

#### Module 1：即时倾诉（核心）

- **匹配逻辑**：情绪标签 → 倾听者擅长领域 → 评分排序 → 在线优先
- **对话限时**：20分钟（双方同意可延长至30分钟）
- **安全机制**：
  - 敏感词检测（自残/自杀 → 弹出危机干预资源 + 通知管理员）
  - 举报/拉黑（一经核实永久设备封号）
  - 对话"阅后即焚"——双方无法回看历史记录
  - 禁止交换任何联系方式

#### Module 2：倾听者体系

- **招募**：心理学专业学生、公益志愿者、互助转化用户
- **培训**：30分钟线上课程（积极倾听 / 避免评判 / 危机识别与上报）
- **认证**：通过模拟对话考核后上岗
- **激励**：
  - 暖心值等级系统
  - 兑换免费倾诉次数
  - 公益服务证书（适合学生简历）
- **AI辅助**：实时向倾听者侧推送回应建议（仅倾听者可见）

#### Module 3：AI兜底

- 无真人在线时自动接管
- 明确告知"我是AI倾听助手"
- 核心能力：情绪识别 + 共情回应 + 引导式提问
- 检测严重情绪时提示转接真人或外部资源

#### Module 4：情绪轨迹（轻功能，条件展示）

- 每次对话后情绪评分记录
- **条件展示策略**（避免下降曲线加剧负面情绪）：
  - **趋势好转时**：展示情绪曲线（正向激励，让用户看到自己在变好）
  - **趋势下降时**：隐藏曲线，改为"行动重构"视角——展示用户主动倾诉次数、被倾听时长、自助练习次数，强调"你一直在照顾自己"，同时温和引导专业资源
- 不做社交化、不做内容Feed

---

## 四、交互设计原则

1. **极简入口**：打开 → 选情绪 → 开始倾诉，3步以内
2. **去社交化**：无好友列表、无动态、无Feed，每次都是全新陌生人
3. **去身份化**：无头像、无个人主页、不可加好友、不留联系方式
4. **安全感优先**：深色UI、柔和色调、对话阅后即焚
5. **限时对话**：创造珍贵感，降低倾听者心理负担

## 五、商业模式

### 定价策略

```
免费层：
  - 每天1次免费真人倾诉（排队匹配）
  - AI倾诉无限次

临时通行证（适合偶尔需要、不想订阅的用户）：
  - 日卡 ¥5：当天无限次真人倾诉 + 优先匹配
  - 周卡 ¥19：7天内无限次真人倾诉 + 优先匹配
  - 到期自动停止，不自动续费（降低付费心理门槛）

月度会员（¥29-49/月，长期用户最划算）：
  - 无限次真人倾诉
  - 优先匹配（减少等待）
  - 选择倾听者风格标签（暖心/理性/幽默）
  - 对话可延长至30分钟
```

**设计思路**：情绪需求是波动性的——用户可能某一周特别难熬但平时不需要。日卡/周卡降低了"为了一次倾诉被锁定月费"的心理阻力，同时也是月度会员的转化漏斗入口。

### 增长路径

```
阶段一：MVP验证（微信小程序）
  - 自建10-20人倾听者团队
  - 验证：对话完成率 > 70%、情绪好转率 > 60%、7日回访率 > 20%

阶段二：小规模付费验证（不融资）
  - 免费AI兜底 + 付费真人倾诉
  - 验证用户付费意愿
  - 月流水能否覆盖运营成本

阶段三：选择方向
  A) to C 会员制——付费验证成立 → 独立App
  B) to B 企业EAP——为企业员工提供匿名倾诉服务
  C) 高校合作——校园心理中心课外支持补充
```

## 六、风险与对策

| 风险 | 对策 |
|------|------|
| 用户倾诉自杀/自残 | 关键词检测 → 强制弹出24小时热线 + 后台预警 + 管理员介入 |
| 倾听者质量不一 | 最低培训门槛 + 评分淘汰 + AI实时辅助提示 |
| 供给侧冷启动 | 先自建核心团队覆盖晚间高峰时段 |
| 政策合规 | 明确定位"陪伴倾诉"非"心理咨询/治疗"，规避医疗资质要求 |
| 留存天花板 | 互助机制（倾诉者→倾听者转化）+ 轻量情绪日记 |
| 商业化天花板 | 不依赖VC，走自盈利 + to B双轨 |

## 七、技术选型（MVP）

- **前端**：微信小程序（冷启动最优选）
- **后端**：Node.js + WebSocket（实时聊天）
- **匹配系统**：简单队列 + 标签过滤
- **AI兜底**：大模型API（文心/通义/DeepSeek）+ 安全过滤层
- **内容安全**：第三方审核API + 关键词规则引擎
- **数据库**：PostgreSQL + Redis（会话状态）

## 八、核心指标

| 指标 | 目标 | 含义 |
|------|------|------|
| 对话完成率 | > 70% | 匹配后完成完整对话的比例 |
| 情绪好转率 | > 60% | 对话后情绪评分高于对话前的比例 |
| 7日回访率 | > 20% | 首次使用7天内再次回来的比例 |
| 倾听者满意度 | > 4.0/5.0 | 倾诉者对倾听者的评分 |
| 付费转化率 | > 5% | 免费用户转付费的比例 |
| 月流水/人 | > ¥35 | 付费用户月均消费 |
